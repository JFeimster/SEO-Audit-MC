# Verification Prompt

## Objective

Verify whether a representative URL for **Distilled Funding by Moonshine Capital** passes Phase 0 quality gates.

## Inputs Required

- URL
- page type
- title tag
- meta description
- canonical
- indexation state
- visible trust anchors
- schema found
- notes on crawl behavior

## Checks

- brand naming consistency
- title and meta accuracy
- canonical correctness
- schema fit
- trust/disclosure adequacy
- crawl/index alignment

## Output Format

- pass or fail
- findings by category
- fixes required before publish approval
- severity
- recommended next step
