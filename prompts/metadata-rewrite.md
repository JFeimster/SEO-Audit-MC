# Metadata Rewrite Prompt

## Objective

Rewrite page metadata for **Distilled Funding by Moonshine Capital** so it is clear, trustworthy, and aligned with the approved brand rule set.

## Inputs Required

- current URL
- current title tag
- current meta description
- page purpose
- primary keyword or topic
- canonical target
- disclosure or trust notes

## Instructions

1. Keep the brand naming consistent with `docs/brand-rules.md`.
2. Do not promise outcomes that are not supported.
3. Improve clarity and search intent alignment.
4. If the page should not remain indexed, say so instead of forcing a rewrite.
5. Flag if the title or description conflicts with the canonical strategy.

## Output Format

- recommendation: keep, rewrite, noindex, merge, or redirect
- rewritten title tag
- rewritten meta description
- rationale
- risks or blockers
