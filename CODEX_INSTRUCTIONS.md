# Codex Instructions

## Repo Goal

Use this repository to stabilize SEO operations for **Distilled Funding by Moonshine Capital** before any scaled page production.

This is an ops and governance repo, not an app repo.

## Source-of-Truth Docs

Read these first before adding or changing artifacts:

1. `README.md`
2. `TASKS.md`
3. `docs/phase-0-triage.md`
4. `docs/brand-rules.md`
5. `docs/pseo-source-of-truth-schema.md`
6. `docs/page-blueprints.md`

## Working Rules

- Treat Phase 0 stabilization as higher priority than content generation.
- Use the bridge brand exactly as documented.
- Keep outputs operator-friendly and editable.
- Prefer markdown, CSV, JSON, and small TypeScript utilities.
- Fail loudly when required canonical, disclosure, or schema inputs are missing.
- Preserve existing repo content and update in place instead of duplicating files.

## Build First

- Brand normalization controls
- Crawl and index control artifacts
- URL inventory and action queue maintenance
- Trust and disclosure remediation guidance
- Schema governance starters
- Verification workflows for 5 representative URLs

## Do Not Build First

- Frontend app code
- Next.js pages
- Large batches of SEO pages
- Invented redirect targets
- Live production copy for unknown URLs
- Fully automated generation pipelines before controls are approved
